Translate Cells dialog

Allows you to translate a selected set of text strings from one language to one or more other languages.

This dialog appears when you right-click a text string (or set of selected text strings) in the Android Localization Files Editor and select Translate cell(s) from the menu that appears. It allows you to specify how all of the selected text strings are to be translated to one or more new languages.

Important: In order to use this feature you must have configured the editor with a key you have previously obtained from Google.
Translator The translation service to be used to translate the selected set of text strings to the new language(s).
From The language the selected strings are in; the translation service translates from the language indicated here to the target language(s).
To The target languages: the languages into which the selected strings are to be translated. Note that you can only translate into languages for which there already exists a localization file (and therefore there is a corresponding column shown in the Android Localization Files Editor).
